# premium-portfolio

## Project layout (single folder)

All source files have been moved into `unified_project`:

- `unified_project/backend` - Express + MongoDB backend
- `unified_project/frontend` - React + Vite frontend

## Run

Backend:
- cd unified_project/backend
- npm install
- npm start (or nodemon server.js)

Frontend:
- cd unified_project/frontend
- npm install
- npm run dev
